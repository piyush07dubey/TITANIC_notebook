# 🚢 Titanic Survival Prediction

## 📌 Project Overview
This project predicts whether a passenger survived the Titanic disaster using classification models.

Focus Areas:
- Missing Value Handling
- Feature Engineering (FamilySize, IsAlone)
- Encoding & Scaling
- Model Comparison
- Accuracy & Precision/Recall Reporting

---

## 📊 Dataset Features

- Pclass
- Sex
- Age
- SibSp
- Parch
- Fare
- Embarked
- Engineered Features (FamilySize, IsAlone)

Target:
- Survived (0 or 1)
